  PdfDest Object
PdfDest represents a destination. This object is creatable via PdfPage's CreateDest method and also PdfDocument's CreateDest method. An instance of the PdfDest object is only created to be passed to other objects' methods as an argument. It has no intrinsic properties of its own. For more information on destinations, see Section 10.1 - Destinations.
